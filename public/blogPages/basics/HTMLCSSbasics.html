Part 1: Intro to HTML

The summary:

HTML stands for HyperText Markup Language. It’s the first building block for a webpage.  It’s simple text instructions that tell your browser what to show to the viewer.  If you were to think of a webpage as a construction project, then the HTML would be all of the construction materials.  

A quick introduction:

Create a new notepad file on your computer.  Copy/paste this inside:

<html>

<head>

<title>Demonstration</title>

</head>

<body>

<p>THIS IS TEXT ON THE PAGE</p>

</body>

</html>

Save it as index.html, and open the file with your browser.  You should see something similar to this:

A little more detail:

The HTML language, at its core, is made up of tags.  A tag is just a word (or letter) between some angle brackets “<>”.  Tags (almost) always come in pairs (one that starts the tag, and another that ends it), so that anything that belongs to them can be placed between them.  Let’s describe what they mean:

  • <html></html>:  These tags are mandatory.  They go at the front & back of your file so that your browser knows that everything in between is HTML code.
  • <head></head>:  These tags are also mandatory.  They tell your browser some of the things that don’t go directly on the page, such as the title, or what language will be on display.  
  • <body></body>:  The last of the mandatory tags.  These contain all the things that actually go on the page.  
  • <title></title>:  These tags describe what goes on the tab at the top.  It’s the title of your document!
  • <p></p>:  “P” stands for Paragraph.  You could have multiple tags like this all over your page if you wanted multiple paragraphs!  

There are many more tags than just those.  If you want to look up some more, I’d recommend these for beginners:

  • <h1></h1>, <h2></h2>, … <h6></h6>
  • <ul><li></li></ul>
  • <ol><li></li></ol>
  • <strong></strong>, <em></em>
